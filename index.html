About this project:

Dash is a project led by Olga Baysal from the University of Waterloo to help provide Mozilla developers with customized dashboards that summarize their current Bugzilla issues. The project was started after investigating the Mozilla Anthropology interviews. The project is in active development; if you have any questions, comments, or suggestions, please get in touch.
